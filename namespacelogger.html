Enumeration Type Documentation

◆ Level

enum logger::Level

Levels for filtering messages based on importance.

Enumerator
Verbose 

Used for low-level debugging information. When logging to console, messages with this level will be sent to cout.

Debug 

Used for debugging information. When logging to console, messages with this level will be sent to cout.

Info 

Used for important information. When logging to console, messages with this level will be sent to cout.

Warning 

Used for logging non-critical problems. When logging to console, messages with this level will be sent to cerr.

Error 

Used for logging critical problems. When logging to console, messages with this level will be sent to cerr.

Off 

Used for disabling outputs.

Enables the logger with the given minimum Levels. Call again to change these settings. This results in a new log file being generated, if file logging is enabled.

Parameters

fileLevelMinimum Level for logging to file. Log messages with a lower Level won't be written to the log file.
